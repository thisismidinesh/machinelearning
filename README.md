## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Explore topic folders for specific ML techniques

## Topics Covered

- **Regression**: Linear, polynomial, and advanced regression methods
- **Classification**: Binary/multi-class classification algorithms
- **Clustering & Dimensionality Reduction**: Unsupervised learning techniques
- **Data Handling**: Pandas, Numpy operations and preprocessing
- **Visualization**: Plotting results and data exploration
- **Deployment**: Serving ML models in production
- **Testing & Profiling**: Unit tests, memory/time profiling
- **Advanced Topics**: OOP in ML, synthetic data, business optimization

Each topic folder contains `tutorial.ipynb` with examples and starter code.
